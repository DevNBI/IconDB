### Introduction
Welcome to ***IconDB***, a collection of icons from various sources. These icons can be used for design as well as development or whatever you may need them for. The goal of ***IconDB*** is to provide icons for free and without daily download limits or subscription requirements.

---

### Icon Sources
***IconDB*** includes icons from the following sources:  
- [Icons8](https://icons8.com)  
- [FlatIcon](https://flaticon.com)  
- [Google Fonts](https://fonts.google.com/icons)  
- [Figma](https://www.figma.com/community/icons?resource_type=files)  
- [Iconify](https://icon-sets.iconify.design/)  
- [NBIcons](https://github.com/devnbi/IconDB)

We thank all of the editors, designers, developers and so on that have created the icons found on our website. It's important to give attribution so please do the same if using icons found on ***IconDB***.

---

### Contributions
If you have your own icons or any that you'd like us to add to this repository, feel free to suggest or provide them. We will attribute anyone who helps with the growth of this project.
